# react-firebase-user-account-system

This is a boilerplate for starting a React user account system authenticating using Firebase authentication.

Note: Update the Firebase config file with your API key and project ID before compiling the project.

See also: [react-mobx-typescript-boilerplate](https://github.com/rokoroku/react-mobx-typescript-boilerplate)

## Installation

```
$ yarn add
```

## Running

```
$ yarn watch
```

## Build

```
$ yarn start
```

## Deploy to Firebase

```
$ yarn run predeploy
$ firebase deploy
```

## Format code (using [Prettier](https://github.com/prettier/prettier))

```
$ yarn run prettier
```
